# YouTalk Blog

Учебный фронтенд-проект по макету компании YouTalk. В проекте реализованы две страницы:

- `blog.html` — страница блога;
- `article.html` — страница статьи.

Проект сделан на HTML, CSS без фреймворков. Верстка адаптирована под desktop, tablet и mobile-экраны на основе экспортов из Figma.

## Структура проекта

```text
educational blog/
├── blog.html
├── article.html
├── README.md
├── css/
│   └── styles.css
├── js/
│   └── main.js
├── assets/
│   ├── icons/
│   └── images/
└── design-reference/
```

## Что реализовано

- семантическая HTML-разметка;
- общая шапка сайта для обеих страниц;
- общая нижняя часть сайта для обеих страниц;
- страница блога с фильтрами, карточками статей и кнопкой «Ещё»;
- страница статьи с основным контентом и боковым блоком `aside`;
- адаптивная сетка карточек через CSS Grid;
- мобильное меню-бургер;
- состояния `:hover` и `:focus-visible` для интерактивных элементов;
- БЭМ-подход в именовании классов;
- SVG-иконки и PNG-изображения;
- атрибуты `alt` для изображений;
- доступная навигация через `aria-label`, `aria-expanded`, скрытые подписи и семантические теги.

## Как открыть проект

1. Скачайте архив проекта.
2. Распакуйте его.
3. Откройте файл `blog.html` в браузере.
4. Перейдите на страницу статьи через карточку или откройте `article.html` напрямую.

## Проверка адаптива

В браузере откройте DevTools и проверьте ширины:

- desktop — 1200 px и выше;
- tablet — около 1024 px;
- mobile — около 414 px.

На мобильной версии:

- навигация в шапке заменяется на бургер;
- фильтры блога прокручиваются горизонтально;
- карточки блога становятся в одну колонку;
- боковая колонка статьи переносится под основной контент;
- футер перестраивается в мобильную раскладку.


## Примечание

Файлы из папки `design-reference` добавлены только как ориентиры по макету. В верстке используются изображения и иконки из папки `assets`.
